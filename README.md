# 🐠 OU Aquarium｜觀賞魚圖鑑網站

![封面圖](https://images.unsplash.com/photo-1501004318641-b39e6451bec6?q=80&w=1200&auto=format&fit=crop)

OU Aquarium 是一個以 **觀賞魚種類介紹** 為主題的網站，  
讓使用者能快速查找各式淡水與海水魚種，  
了解牠們的習性、水質需求、食性與飼養難度。  

本網站使用 **HTML + CSS + JavaScript** 製作，  
可在電腦與手機上自動調整版面，並支援深色模式。

## 🧭 專案簡介

這是一個以「展示與教育」為主軸的觀賞魚圖鑑網站，  
結合搜尋與條件篩選功能，幫助新手與愛好者找到適合的魚種。  

你可以依照：
- 淡水 / 海水環境  
- 飼養難度（新手、中階、進階）  
- 性情（溫和、略兇、兇猛）  
快速篩選出理想魚種，並點擊卡片查看詳細資料（體長、pH、水溫、食性等）。
